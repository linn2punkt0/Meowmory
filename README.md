## Installation:
1. Download this repository into htdocs.

2. Open the index.html-file in your browser.

Or:  
  
2. Start Mamp. Open up this repository in your localhost.  
  
    
Or:  
  
1. Go to github-pages-link: https://linn2punkt0.github.io/Meowmory/
